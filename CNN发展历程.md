# CNN发展历程

## 模型架构变迁

| 模型架构 | AlexNet | VGG nets | ResNet |
| --- | --- | --- | --- |
| 架构 | 5Conv+3FC | 19Conv+3FC | 152Conv+1FC |
| 激活函数 | 首次引入ReLU | ReLU | ReLU |  
| 激活抑制机制 | LRN | 发现LRN不能提升性能，弃用 | BN 批归一化取代LRN |
| 池化方式 | 重叠池化缓解过拟合 | 发现重叠池化无效，Max Pooling | 仅保留少量池化操作，更多通过卷积层设置改变维度 |
| 全连接层 | 4096+4096+1000+softmax | 4096+4096+1000+softmax | (GAP)+1000+softmax 极大减少计算量 |  
| GPU | 双GPU通信 | 单GPU | 架构未设计，采用多卡数据并行 |  
| Dropout正则化 | 前两个FC层 | 前两个FC层 | 由于FC减少，BN正则化，过拟合缓解，不需要dropout |  

## 顶级论文实验技巧

| 实验策略 | AlexNet | VGG nets | ResNet |
| --- | --- | --- | --- |  
| 图像数据增强 | 随机裁剪+水平翻转 | 随机裁剪+水平翻转+多尺度对比 | 水平翻转 |  
| 颜色增强 | PCA变换增强 | PCA 变换增强 | 减去RGB均值 |  
| SGD batch size | 128 | 256 | 256 |  
| 初始学习率 | 0.01 | 因为放弃LRN，谨慎调整学习率 0.01 | 0.1 (采用优异的BN，学习率大开大合) |  
| 学习率调整(性能停滞时) | /=10 | /=10 | /=10 |  
| 权重衰减(L2正则化) | 0.0005| 0.0005 | 0.0001 |
| momentum | 0.9 | 0.9 | 0.9 |  
| 权重(卷积核)初始化 | N(0,0.01) | 先用浅层网络预训练，再作为深层对应的初始化;S=384用S=256的初始化 | N(0,0.01) |  
| 测试策略 | --- | 采用卷积层替换fc，Dense Evaluation | --- |
